## 1. 信息架构

```
DropShipFlow
│
├── 浏览器插件 (Chrome Extension)
│   ├── 商品抓取面板 (Popup)
│   ├── 导入选项配置
│   └── 进度通知
│
└── 管理后台 (Admin Dashboard)
    ├── 📊 概览 (Dashboard)
    │   ├── 今日导入统计
    │   ├── 处理中任务
    │   └── 快捷操作
    │
    ├── 📦 商品管理 (Products)
    │   ├── 商品列表 (过滤/搜索/排序)
    │   ├── 商品详情 (翻译/图片/定价预览)
    │   └── 发布到 Shopify
    │
    ├── 🔄 导入任务 (Import Tasks)
    │   ├── 任务列表
    │   ├── 任务详情 (进度/steps/日志)
    │   └── 重新处理 / 取消
    │
    ├── 🏪 店铺管理 (Shops)
    │   ├── 连接 Shopify 店铺
    │   ├── 店铺列表
    │   └── 店铺配置
    │
    ├── 💰 定价规则 (Pricing Rules)
    │   ├── 规则列表
    │   ├── 创建/编辑规则
    │   └── 规则测试
    │
    └── ⚙️ 设置 (Settings)
        ├── 账户信息
        ├── API Keys
        └── 套餐/用量
```

## 3. 核心页面原型

### 3.1 浏览器插件 Popup

#### 3.1.1 布局规格

- **尺寸：** 380 × 520 px (Popup 窗口)
- **背景：** #FFFFFF
- **字体：** Inter / SF Pro Text
- **主色调：** #6366F1 (Indigo-500)

#### 3.1.2 状态 A：未登录 (首次使用 / Token 过期)

```
┌─────────────────────────────────────┐ 380px
│  DropShipFlow  Logo                 │ 48px
├─────────────────────────────────────┤
│                                     │
│              🔌                     │
│                                     │
│      欢迎使用 DropShipFlow           │
│   1688 商品一键上架到 Shopify        │
│                                     │
│   请先登录你的账号以开始使用          │
│                                     │
│   ┌───────────────────────────┐    │
│   │   🔗 登录 / 注册           │    │
│   │   点击后将打开 Web 管理后台  │    │
│   │   admin.dropshipflow.com   │    │
│   │   登录后自动连接插件        │    │
│   └───────────────────────────┘    │
│                                     │
│   没有账号？访问 admin.dropshipflow  │
│                                     │
└─────────────────────────────────────┘
```

**交互：** 点击「登录/注册」→ 自动打开 `admin.dropshipflow.com/login?source=extension`。登录成功后页面自动将 Token 回传给插件（`chrome.runtime.sendMessage`），插件自动切换到状态 B。整个过程用户无需输入任何内容到插件中。

#### 3.1.4 交互行为

| 元素 | 行为 |
|------|------|
| 商品预览卡片 | 点击可展开大图 |
| SKU 数量 | 点击展开 SKU 列表详情 |
| 目标店铺下拉 | 显示已绑定的 Shopify 店铺列表 |
| 定价规则下拉 | 显示用户配置的定价模板 |
| 一键上架按钮 | 提交任务并显示进度动画 |
| 取消按钮 | 关闭 Popup，不执行任何操作 |
| 已登录行 | 显示当前账号，点击「退出」清除 Token 并回到未登录状态 |

### 3.2 插件状态指示

#### 3.2.1 插件图标状态

| 图标 | 含义 | 触发条件 |
|------|------|---------|
| ⚫ 灰色 + 红点 | 未登录 | 插件未连接账号，点击跳转 Web 登录 |
| 🔘 灰色 | 未激活 | 已登录但不在 1688 商品页面 |
| 🔵 蓝色 + 脉冲 | 可抓取 | 已登录且在 1688 商品详情页 |
| 🟢 绿色 | 已抓取 | 数据提取完成，待确认 |
| 🟠 橙色 | 上传中 | 正在提交到后端 |
| ✅ 绿色对勾 | 成功 | 导入任务已创建 |

## 4. 交互规范

### 4.1 响应式断点

| 断点 | 宽度 | 适用场景 |
|------|------|---------|
| 插件 Popup | 380px | 固定宽度，无需响应式 |
| 管理后台 Tablet | ≥768px | 折叠侧边栏为汉堡菜单 |
| 管理后台 Desktop | ≥1024px | 完整布局，侧边栏展开 |

### 4.2 组件状态

| 组件类型 | 状态 | 视觉表现 |
|---------|------|---------|
| 按钮 Primary | Default / Hover / Active / Loading / Disabled | Indigo 渐变 / 加深 / 内阴影 / 旋转图标 / 灰色 |
| 按钮 Secondary | 同上 | 边框样式 |
| 输入框 | Default / Focus / Error / Disabled | 灰边框 / Indigo 边框 / 红色边框 + 提示 / 灰底 |
| 下拉选择 | Default / Open / Selected | 同输入框 |
| 状态标签 | 草稿 / 已发布 / 失败 / 处理中 | 黄 / 绿 / 红 / 蓝 |
| 进度条 | 0-100% | Indigo 渐变填充 + 百分比文字 |
| Toast 通知 | Success / Error / Warning / Info | 绿 / 红 / 黄 / 蓝 左侧色条 |

### 4.3 动画规范

| 动画 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| 按钮 hover | 150ms | ease-out | 颜色/阴影过渡 |
| 骨架屏加载 | 1.5s | linear | 内容加载占位 |
| 进度条更新 | 300ms | ease-in-out | 百分比变化 |
| 弹窗出入 | 200ms | ease-out | Popup/Modal 显隐 |
| 列表项入场 | 300ms | ease-out + 50ms stagger | 列表刷新 |
| 脉冲指示 | 2s | ease-in-out 循环 | 插件图标可抓取状态 |

### 4.4 无障碍 (A11y)

- 所有交互元素支持键盘导航 (Tab/Enter/Escape)
- 图标按钮有 `aria-label`
- 表单有 `<label>` 关联
- 颜色不是传达信息的唯一方式（搭配图标/文字）
- Focus visible 样式明显
